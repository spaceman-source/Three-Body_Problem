# Three-Body-Problem

This repository consists of numerical simulations of the Collinear (Euler) solution, Triangular (Lagrange) solution, and the Checiner, Montgomery solution, Sitnikov Solution and the Yin-Yang solution.
These files are written in python utilizing primarily matplotlib, vpython and numpy libraries. All these files have a very similar structure utilizing the same mechanics equation for calculating trajectories.
I plotted most of them in two dimensions for simplicity but also because the trajectories excluding (Sitnikov) have little motion in three dimensions.
